# LimeStudio: The Visible Compiler for Audio Logic

LimeStudio is a Rust-based audio framework designed for professional-grade reliability and visual transparency.

## Squeezer IDE: The Heart of Development
At the center of LimeStudio is **Squeezer IDE**. It is a visual integrated development environment where you build DSP graphs, define declarative UIs, and audit real-time signals in a single, unified space.

- **Live Graph Patching**: Build and reconfigure DSP using VPL (Visual Programming Language) without stopping the audio.
- **UI Hot-Reloading**: Preview your declarative UI instantly with live parameter feedback.
- **Forensic Inspection**: Audit signal flow and compiler inferences in real-time to ensure absolute trust in your code.
