# LimeStudio 入門：自分だけの「音の楽器」を作ろう

このガイドは、「プログラミングはよくわからないけど、自分だけのシンセサイザーやエフェクトを作ってみたい」という人のためのものです。LimeStudio の IDE **Squeezer** を使えば、数学や難しいコードを知らなくても、プロと同じ品質のプラグインが作れます。

---

## 1. Squeezer を開こう

まずは Squeezer を起動しましょう。ここがあなたの「工房」になります。

```bash
cargo run -p squeezer
```

## 2. 音の「流れ」を作ろう (VPL)

中央の広い画面で、右クリックして「音の部品（ノード）」を呼び出します。

- **Oscillator (オシレーター)**: 音の源です。
- **Filter (フィルター)**: 音を削って形を整えます。
- **Output (出力)**: スピーカーへ音を送ります。

これらを線でつなぐだけで、音が鳴り始めます。間違えても壊れることはありません。いろいろな組み合わせを試してみましょう。

## 3. かっこいい「見た目」を作ろう (UI)

音が決まったら、それを操作する「つまみ（ノブ）」を配置します。

Squeezer の「Design」ボタンを押すと、画面上にノブを配置できます。配置したノブを、VPL の画面にある「フィルターの強さ」などにドラッグしてつなげるだけで、操作できるようになります。

## 4. プロの品質を保証する「監査官」

あなたが作ったプラグインは、裏側で「Lime 監査官」がチェックしています。
- つまみの位置がズレていないか？
- 音が割れていないか？
- 壊れやすい計算をしていないか？

監査官が「OK」と言えば、それは世界中のプロが使うプラグインと同じ信頼性を持っている証拠です。

## 5. 世界へ出荷しよう！ (Release)

完成したら、右上の **BUILD** ボタンを押すだけです。

`LimeBuilder` が自動的に Apple の証明書などを準備して、あなたの Mac や iPad で今すぐ使える「本物のプラグイン」を書き出します。

---

### 魔法を信じるな、確信を持て

LimeStudio は「魔法」で音を鳴らしているのではありません。あなたが引いた「線」と、配置した「つまみ」の因果関係が、そのまま音になっています。

自分だけの「音の法」を作り、世界を驚かせましょう！
