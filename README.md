# LimeStudio: The Visible Compiler for Audio Logic

LimeStudio is a high-performance audio framework designed for professional-grade reliability and visual transparency.

## Squeezer IDE
The heart of LimeStudio is **Squeezer IDE**—a visual environment for live DSP patching, declarative UI design, and forensic signal inspection.
